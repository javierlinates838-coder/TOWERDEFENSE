# Rootbound

A polished, offline-first tower defense game about growing a connected defensive ecosystem.

## The idea

Rootbound replaces random loot and fixed tower upgrades with a deterministic living-network system:

- Nearby towers connect their roots automatically.
- Every different species linked to a tower grants it +12% attack speed.
- Four complementary species support rapid fire, slowing, burst damage, and chain attacks.
- Kills charge **Overgrow**, an eight-second network-wide surge.
- Ten hand-balanced waves culminate in the Rift Sovereign boss.

The complete game runs in the browser with no dependencies, accounts, ads, or microtransactions.

## Run locally

```bash
npm start
```

Then open [http://localhost:4173](http://localhost:4173).

## Controls

- `1`–`4`: select a seed
- Click a glowing node: plant
- Click a tower: inspect, evolve, or prune it
- `Space`: pause
- `F`: change speed
- `R`: activate Overgrow when charged
